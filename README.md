# Oasis Infobyte - Data Science Internship (OIBSIP)

This repository contains the machine learning models, exploratory data analysis, and predictive workflows developed during my virtual Data Science internship with Oasis Infobyte.

*Note: All core analysis and code architecture were written and tested locally using Jupyter Notebooks over the 4-week program. The finalized scripts and documentation were committed to GitHub upon successful optimization.*

## 📁 Repository Structure
* `Task 1`: Iris Flower Classification Model (Supervised learning)
* `Task 3`: Car Price Prediction Engine (Regression algorithms)
* `Task 4`: Email Spam Recognition Pipeline (Natural Language Processing)

## 📄 Internship Credentials
* **Status:** Completed Successfully
* **Documentation:** Official offer letter included in the root directory.

## 🛠️ Data Science & ML Stack
* **Language:** Python 3
* **Libraries:** Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn
* **Development Environment:** Jupyter Notebooks 

## 📊 Project Overviews

### 🔬 Task 1: Iris Flower Classification
* **Objective:** Built a classification model to categorize iris flower species using sepal and petal measurement datasets.
* **Key Implementations:** Data cleaning, supervised classification metrics, model accuracy optimization.

### 🚗 Task 3: Car Price Prediction
* **Objective:** Developed a regression pipeline to accurately predict used car market valuations based on vehicle attributes.
* **Key Implementations:** Exploratory Data Analysis (EDA), numerical feature engineering, linear/tree regression modeling.

### ✉️ Task 4: Email Spam Recognition
* **Objective:** Programmed an NLP classification engine to analyze text strings and distinguish between valid emails (ham) and spam messages.
* **Key Implementations:** Text preprocessing, vectorization paradigms, binary text classification.
